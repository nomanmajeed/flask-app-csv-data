1. Open cmd and redirect to the project folder.

2. At the project folder root i-e: C:\Users\user_name\Desktop\Project Folder (in case you have folder at desktop)
    type the command and press enter to execute the file: python app.py

3. You will see the following output at cmd:
        * Serving Flask app "app" (lazy loading)
        * Environment: production
        WARNING: This is a development server. Do not use it in a production deployment.
        Use a production WSGI server instead.
        * Debug mode: on
        * Restarting with stat
        * Debugger is active!
        * Debugger PIN: 192-495-685
        * Running on http://127.0.0.1:5000/ (Press CTRL+C to quit)

4. Your program is running on localhost:5000.

5. Open any browser and type the url localhost:5000 or 127.0.0.1:5000 and you will see a web page

6. Now this webpage will have a inputfiled for url and a submit button. Enter file url and submit it you will see a flash message
    and a csv file will be generated at the root of your project folder.
